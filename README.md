# Keyforge 3D

## Stack

- TypeScript
- React
- React Three Fiber (R3F)
- Tailwind CSS v4

## Tools Used

- npm
- Vite
- Blender

## Description

This project was built for educational purposes to explore React Three Fiber. It features an interactive 3D keyboard model built from scratch in Blender and exported into the `public/` folder as `keyboard.glb`.

> **Note:** This is a non-commercial project, and the 3D keyboard model is not a real-life product.
